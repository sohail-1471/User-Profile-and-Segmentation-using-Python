# User Profile & Segmentation Analysis (Python)

## 📌 Project Overview

This project focuses on analyzing user behavior and performing segmentation using Python. The objective is to group users based on demographic, behavioral, and engagement metrics to better understand different audience segments.

The insights derived from this analysis can be used for targeted marketing, personalized recommendations, and improving user engagement strategies.

---

## 📊 Dataset Description

The dataset includes a wide range of user attributes:

* **Age** – User's age
* **Gender** – User's gender
* **Location** – Geographic location
* **Language** – Preferred language
* **Education** – Educational background
* **Likes and Reactions** – Engagement metrics
* **Device Usage** – Type of device used (mobile, desktop, etc.)
* **Time Spent Online** – Duration of user activity
* **Followed Accounts** – Number or type of accounts followed
* **Click Through Rate (CTR)** – Ad engagement metric
* **Conversion Rate** – Percentage of users completing desired actions
* **Ad Interaction Time** – Time spent interacting with ads
* **Income Level** – User income category
* **Top Interests** – User interests and preferences

---

## 🎯 Objectives

* Segment users into meaningful groups based on behavior and demographics
* Identify high-value user segments
* Analyze engagement patterns across different user groups
* Understand factors influencing CTR and conversion rates
* Discover relationships between income, interests, and user activity

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Pycharm

---

## 🔍 Key Analysis Performed

### 1. Data Preprocessing

* Handling missing values and outliers
* Encoding categorical variables
* Feature scaling for clustering

### 2. Exploratory Data Analysis (EDA)

* Demographic distribution analysis
* Engagement metrics visualization
* Device usage trends

### 3. User Segmentation

* Clustering using algorithms like K-Means
* Identifying distinct user groups
* Profiling each segment based on key attributes

### 4. Behavioral Analysis

* CTR and conversion rate comparison
* Ad interaction behavior across segments
* Time spent vs engagement correlation

### 5. Visualization

* Cluster plots for segmentation
* Heatmaps for correlation analysis
* Bar charts and pie charts for distribution insights

---

## 📈 Insights & Findings

* Distinct user segments emerge based on engagement and demographics
* High-income users may show higher conversion rates
* Device usage influences user behavior and engagement patterns
* Certain interest groups have significantly higher interaction rates
* Time spent online is strongly linked with user activity and conversions
